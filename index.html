    📦 교재통합형 (기본/권장)

    수강료 + 교재비 (2단 분리)
    재료비를 교재비에 합산하는 보편적 방식

    🍱 교재분리형 (심화)

    수강료 + 교재비 + 재료비 (3단 분리)
    항목을 엄격히 분리하여 고지하는 학교용

    ⚙️ 시스템 환경 설정 (글로벌 룰)
    1. 공제 연산 유형

    우선순위가 높은 강좌 하나를 0원이 될 때까지 집중 공제한 후 다음 강좌로 이동

    스마트 출석부 양식(태그) 사용 가이드
    학교에서 기존에 사용하시던 엑셀 파일 빈칸에 약속된 기호(태그)를 적어 올리면, 시스템이 수강생 데이터를 알아서 채워 넣습니다.
    1. 시스템 태그 [ ] (대괄호 1개)

    시스템에 저장된 분기/강좌/학생 정보를 자동으로 불러오는 태그입니다.

    태그명출력 예시설명
    [분기]1현재 작업 중인 분기 숫자
    [강좌명]로봇과학해당 시트의 방과후 강좌명
    [학적]3-2-15학년-반-번호를 한 번에 출력
    [이름]홍길동수강생의 이름
    [연락처]010-123-4567마스터 엑셀을 올렸을 때만 자동 매칭
    [귀가방법]도보(학원)마스터 엑셀을 올렸을 때만 자동 매칭
    [학년], [반], [번호] 등 개별 태그도 지원합니다.
    2. 사용자 정의 빈칸 태그 [[ ]] (대괄호 2개)

    결재자, 강사명 등 시스템에 없는 내용을 파일 생성 전 화면에서 직접 타이핑할 수 있게 빈칸을 뚫어주는 태그입니다.

    예시: [[강사명]], [[교육기간]], [[작성일자]] 등을 자유롭게 적어보세요.
    ※ 한 번 입력한 값은 저장(캐싱)되어 다음번 출력 시 자동으로 채워집니다.
    양식 작성 시 핵심 주의사항 (스마트 앵커)
    • 반드시 넉넉한 인원수만큼 표를 길게 그려두세요. (예: 최대 정원 30명 표)
    • 시스템은 [학적]이나 [이름] 태그가 적힌 줄을 명단 시작점(1번 학생)으로 인식합니다.
    • 표의 가장 첫 번째 줄(1번 자리)에만 태그를 적고 나머지는 비워두세요.
    • 실제 학생 수가 20명이면, 20명만 채우고 남은 10칸의 빈 줄은 시스템이 자동으로 엑셀에서 숨김 처리하여 하단의 결재란을 위로 예쁘게 끌어올립니다!
